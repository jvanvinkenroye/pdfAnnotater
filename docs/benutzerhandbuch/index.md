# Benutzerhandbuch

Dieses Handbuch erklärt die Nutzung des PDF Annotators Schritt für Schritt.

## Typischer Ablauf

```
1. Einloggen / Registrieren
        ↓
2. PDF hochladen
        ↓
3. Metadaten eingeben (optional)
        ↓
4. Seiten durchblättern & annotieren
        ↓
5. Exportieren (PDF oder Markdown)
```

## Abschnitte

| Abschnitt | Inhalt |
|---|---|
| [PDF hochladen](upload.md) | Upload-Dialog, Drag & Drop, Metadaten |
| [Viewer & Navigation](viewer.md) | Split-Screen, Seitennavigation, Zoom, Shortcuts |
| [Annotationen](annotationen.md) | Notizen erfassen, Auto-Save, Seiten verwalten |
| [Export](export.md) | Annotiertes PDF, Markdown, Original, Backup |
